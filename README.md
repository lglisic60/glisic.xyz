# glisic.xyz

Personal research site of Leo Glisic — mechanism design and network goods for Ethereum.

Plain HTML + CSS, no build step, no dependencies. Served via GitHub Pages at
https://glisic.xyz.

## Structure

- `index.html` — homepage / publication index
- `ppn/` — The Private Payment Network (article)
- `papers/` — versioned PDFs (`*-v1.pdf`, `*-v2.pdf`, … — never overwrite a cited file)
- `style.css` — the only stylesheet
- `CNAME` — custom-domain binding for GitHub Pages

## URL policy

Publication URLs are permanent. Once a path or PDF filename has been cited
anywhere, it does not change; revisions are published alongside as new versions.
